Design Notes

  • Discovery: Shows agent extracting from SAP systems (15s auto-advance)
  • Briefing: AE approval gate with strategic context (user-controlled)
  • Generation: Progress bar + collapsible reasoning (90s auto-advance)
  • Results: Real PDF downloads + functional refinement loop
  • Styling: Dark theme, Apple design system, smooth animations
  • Data: Love's facts match FINANCIAL-AUDIT.md (897 orders, $7.6M, ROI 215%/227%/227%)
